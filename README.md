### local-development' Local Development

To run this project locally:

1.  Clone the repository: `git clone <your_repo_url>`
2.  **Backend Setup:**
    ```bash
    cd server
    npm install
    # Create a .env file with MONGO_URI and JWT_SECRET
    npm start
    ```
3.  **Frontend Setup:**
    ```bash
    cd client
    npm install
    npm run dev
    ```
